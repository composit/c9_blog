about

Composition9 Design, LLC is Matt Rose. I am a professional web application developer based in Fort Collins, Colorado. I work on a wide variety of projects. Feel free to send me your project proposals. If I do not think my skills are the right match for your project, I might be able to point you in the right direction.

I build highly accessible, highly functional web applications and browser-based information systems.

My focus is server-side scripting and standards-compliant web development. I mainly code using Ruby on Rails, relational databases (Postgres, MySQL), Javascript (AJAX), CSS and HTML, and am well-versed in many other technologies, including, but not limited to: versioning (Git), NoSQL datastores (MongoDB), BDD Testing (mostly Rspec), Server admin (Apache, Nginx), some Linux administration, some Coffeescript, SCSS, HAML, Backbone.js, etc.
